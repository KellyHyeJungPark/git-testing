-----
* git clone 원격저장소주소
* git add */./특정파일
* git commit -m "추가할 메세지"
* git push: git bash에서 git commit -m 으로 추가한 메세지 저장
* git pull: 원격저장소의 변경내용을 로컬로 가져오게 하는 기능
=> VSC 메뉴 -> source code ->  source control repositories 에서 로컬로 불러온 파일 확인 가능
